##Descripción del Proyecto

Este repositorio contiene el desarrollo del **Desafío 1 del curso de Python**, donde se aplican conceptos iniciales del lenguaje en un ambiente interactivo mediante Jupyter Notebook.  

El objetivo es aprender de forma práctica cómo funciona Python, desde la creación de variables hasta estructuras condicionales y manejo básico de datos.

---

## Contenidos Cubiertos

- Introducción al entorno Jupyter  
- Variables y tipos de datos  
- Operadores matemáticos  
- Listas y diccionarios  
- Condicionales (`if`, `elif`, `else`)  
- Entrada de datos con `input()`  
- Ejercicio práctico final
